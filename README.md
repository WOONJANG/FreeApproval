# Approval

멀티회사 결재/공지/운영 관리 프로젝트입니다.

## 주요 기능
- 기업가입, 회사코드, 개인 회원가입
- 회사별 문서/회원/공지 격리
- 레벨 기반 순차 결재, 반려/재기안, 기안취소, 휴지통 복구
- 회사 대표의 운영 권한 위임
- 결재선 템플릿, 문서 분류, 참조자/열람자 지정
- 활동 로그, 알림, 결재 독촉, 결재 대기 일수 표시
- 회사별 플랜/이용상태/광고제거 기간/일괄승인 설정
- 공지사항 상단 고정 + 배너 + 팝업
- CSV 기반 엑셀 다운로드
- 본인확인형 비밀번호 재설정
- 모바일 하단 고정 액션 바
- 한국어/영어 전환용 기본 다국어 구조(미구현)

## 설치
1. `approval` 폴더 업로드
2. `config/config.php` DB 정보 수정
3. phpMyAdmin에서 `install.sql` 실행
4. 최상위 관리자 로그인 후 회사별 설정 진행
